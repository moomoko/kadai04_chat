ももとだいのムードトラッカーアプリに、チャット機能、背景選択機能、ハート機能を付け加えて遊び心のあるアプリにしました✍🏻

## ⑤工夫した点・こだわった点

今までずっと同じだった見た目を少し変えました！
1.トップでももだいをタップするとタップしただけハートが出てきます💖ゆらゆらさせたり、ハートの滞在時間、消え方を工夫しました。
2.背景を3種類から選べます！背景の色味はやさしく癒されるようなものを意識しました。波の形も少しずつ違います。
